API Documentation

Guia completo para integração do E-commerce Fullstack

Autenticação

POST /api/v1/login

Use para obter o token de acesso Sanctum.

{ "email": "user@example.com", "password": "password" }

Carrinho & Itens

POST /api/v1/cart-items

Adiciona um produto ao carrinho.

{ "cart_id": "uuid", "product_id": "uuid", "quantity": 1 }

Checkout & Pagamento

Passo a Passo da Integração

  1. Checkout: Envie o cart_id para POST /api/v1/checkout. Isso converterá o carrinho em um Pedido (Order) com status pending.
  2. Pagamento: Receba o order_id retornado e envie para POST /api/v1/payments junto com o método de pagamento.

Implementação no Frontend (Axios)

import axios from 'axios';

const api = axios.create({
    baseURL: 'http://localhost/api/v1',
    headers: { Authorization: `Bearer ${token}` }
});

// Exemplo: Finalizar Compra
const handleCheckout = async (cartId) => {
    try {
        const { data: order } = await api.post('/checkout', { cart_id: cartId });
        const { data: payment } = await api.post('/payments', { 
            order_id: order.id, 
            payment_method: 'pix' 
        });
        alert('Pagamento aprovado!');
    } catch (err) {
        console.error('Erro no checkout', err);
    }
};
